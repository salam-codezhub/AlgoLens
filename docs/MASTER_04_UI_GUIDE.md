# AlgoLens UI/UX Guide

## Vision
Create a premium developer experience inspired by GitHub Copilot, Linear, Raycast, Arc and Vercel.

## Design Principles
- Minimal
- Fast
- Glassmorphism
- Accessible
- Keyboard-first
- Responsive

## Theme
Primary: Electric Blue
Accent: Purple
Success: Green
Warning: Amber
Danger: Red
Background: #0B0F17
Surface: #151B23

## Typography
- Headings: Inter Bold
- Body: Inter
- Monospace: JetBrains Mono

## Navigation
Activity Bar:
- Dashboard
- Analyzer
- Optimization
- Runtime
- Memory
- Security
- AI Chat
- Visualization
- Reports
- History
- Settings

## Dashboard Layout
Header:
- Project
- Current File
- Language
- Scan Status

Cards:
- Overall Score
- Performance
- Complexity
- Runtime
- Memory
- Security

Charts:
- Runtime Trend
- Memory Trend
- Complexity Trend

## Analyzer Screen
Panels:
- AST Summary
- Algorithm
- Functions
- Classes
- Loops
- Complexity
- Suggestions

## Optimization Screen
Two-column layout:
Left:
- Original Code
Right:
- Optimized Code

Bottom:
- Unified Diff
- Accept
- Reject
- Explain

## AI Chat
Features:
- Streaming responses
- Context aware
- Code selection
- Suggested prompts
- Copy response

## Runtime Screen
Show:
- Estimated Runtime
- Measured Runtime
- Benchmark Table
- CPU Trend
- Input Size Selector

## Memory Screen
Show:
- Heap
- Stack
- Peak Memory
- Growth Graph

## Security Screen
Cards:
- Critical
- High
- Medium
- Low

Each issue contains:
- Description
- Severity
- Fix

## Visualization
Use:
- Mermaid
- React Flow
- D3

Generate:
- Flowchart
- CFG
- Call Graph
- Dependency Graph

## Reports
Export:
- PDF
- HTML
- Markdown
- JSON

## Motion
Use Framer Motion.
Animations under 250ms.
No distracting effects.

## Accessibility
- Keyboard navigation
- Focus states
- High contrast
- Screen reader labels

## UI Definition of Done
- Responsive
- Consistent spacing
- Reusable components
- Dark/Light theme
- Accessible
- No visual glitches
